\hypertarget{index_intro}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro}
This is the documentation for the ESP32-\/based Vehicle Control Unit (VCU). The system manages all aspects of electric vehicle control including\+:


\begin{DoxyItemize}
\item Motor control and torque management
\item Battery system monitoring and protection
\item Charging system control
\item Vehicle state management
\item Real-\/time diagnostics and monitoring
\end{DoxyItemize}\hypertarget{index_arch}{}\doxysection{\texorpdfstring{System Architecture}{System Architecture}}\label{index_arch}
The VCU runs on a dual-\/core ESP32\+:
\begin{DoxyItemize}
\item Core 0\+: CAN communication and fast control loops
\item Core 1\+: State management and system monitoring
\end{DoxyItemize}\hypertarget{index_modules}{}\doxysection{\texorpdfstring{Main Modules}{Main Modules}}\label{index_modules}

\begin{DoxyItemize}
\item \doxylink{class_state_manager}{State\+Manager}\+: Vehicle state and safety management
\item \doxylink{class_c_a_n_manager}{CANManager}\+: Communication with vehicle subsystems
\item \doxylink{class_vehicle_control}{Vehicle\+Control}\+: Motor control and driving dynamics
\item \doxylink{class_serial_console}{Serial\+Console}\+: Debug interface and monitoring
\end{DoxyItemize}\hypertarget{index_safety}{}\doxysection{\texorpdfstring{Safety Systems}{Safety Systems}}\label{index_safety}
The VCU implements multiple safety systems\+:
\begin{DoxyItemize}
\item Battery voltage and current protection
\item Temperature monitoring and management
\item Precharge sequence control
\item Watchdog monitoring
\item Error detection and handling
\end{DoxyItemize}\hypertarget{index_config}{}\doxysection{\texorpdfstring{Configuration}{Configuration}}\label{index_config}

\item Battery limits and thresholds
\item Motor performance parameters
\item Temperature limits
\item Control response curves 
\end{DoxyItemize}
