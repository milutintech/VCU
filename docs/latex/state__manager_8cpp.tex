\doxysubsection{Detailed Description}
Implementation of the vehicle state management system. 

Manages the vehicle\textquotesingle{}s operational states and transitions\+:
\begin{DoxyItemize}
\item STANDBY\+: Low power state with minimal systems active
\item RUN\+: Normal driving operation with motor systems enabled
\item CHARGING\+: Battery charging state with charger enabled
\end{DoxyItemize}

Handles critical safety systems including\+:
\begin{DoxyItemize}
\item Battery precharge sequence
\item Contactor control
\item Cooling system management
\item System wake-\/up and sleep control 
\end{DoxyItemize}
