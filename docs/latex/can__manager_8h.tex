\doxysubsection{Detailed Description}
CAN Communication Manager for Vehicle Control System. 

Manages all CAN bus communication between vehicle subsystems\+:
\begin{DoxyItemize}
\item Battery Management System (BMS)
\item Drive Motor Controller (DMC)
\item Battery Switch Controller (BSC)
\item Network Load Gateway (NLG) for charging
\end{DoxyItemize}

Handles message scheduling, data parsing, and system state coordination
